#  Student Performance Analyzer Guide

This repository contains a beginner-friendly explanation of the **Student Performance Analyzer** project developed using **Python**, **Streamlit**, **Pandas**, and **NumPy**.

The guide explains the important statements in the program with simple examples so beginners can understand how the project works.

## Topics Covered

- Importing Libraries
- Streamlit Page Configuration
- Session State
- Sidebar Menu
- Add Student
- View Students
- Analytics
- Grade Report
- Search Student
- Top 3 Students

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy

## Purpose

This guide was created to help students understand the logic behind each section of the program instead of simply copying the code.


## How to run the program
- First create a file
- Add this code
- Then click on breadcrumb
- Write cmd and press enter so it will automatically open command
- On command write "streamlit run (file name) ex- my file name is stu_performance analyzer so i will write "streamlit run stu_performance_analyzer.py"
- Done
